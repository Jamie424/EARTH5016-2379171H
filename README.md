# EARTH5016-2379171H

This code is a numerical thermal model of a 2D cross section in Helmsdale. The domain includes a basement gneiss, granite body, with overlain sediments and fluvial deposits with a normal fault 

To reproduce figures from report, open the relevant matlab file within the 'usr' folder, such as Thermal_ref, and run the file. The output figures will then be saved to the out folder in the relevent subfolder automatically. 

All parameters are kept consistent in each Thermal simulation apart from the commented parameters at the top of each script. 

For Verification, run the two files run_convtest_dt and run_convtest_dx. The validation1 file is a test file where parameters have been modified for validation purposes.
